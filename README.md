# ICI Factory — الشركة الدولية للصناعات

موقع إلكتروني ثابت ثنائي اللغة (عربي RTL / إنجليزي) للشركة الدولية للصناعات (ICI) — مصنّع الملح في قطر.

A bilingual (Arabic RTL / English) static website for International Industries Company (ICI), a salt manufacturer in Qatar.

## 📂 البنية / Structure

```
icifactory/
├── index.html          # الرئيسية / Home
├── about.html          # عن الشركة / About
├── products.html       # المنتجات / Products
├── contact.html        # تواصل / Contact
├── disclaimer.html     # إخلاء المسؤولية / Disclaimer
├── privacy.html        # الخصوصية / Privacy
├── thanks.html · 404.html
└── assets/
    ├── css/style.css   # كل التنسيقات / All styles
    ├── js/main.js      # اللغة + السلايدر + القوائم
    └── images/
        ├── brand/      # شعار ICI
        ├── clients/    # شعارات العملاء / Client logos
        ├── products/   # صور المنتجات (WebP)
        └── factory/    # البانر وصور المصنع (WebP)
```

## 🚀 النشر على Hostinger / Deploy

ارفع محتويات المستودع (ما عدا `README.md` و `.git`) إلى `public_html` مباشرة في الجذر.
Upload repo contents to `public_html` root (not a subfolder).

## 🖼️ الصور / Images

الصور بصيغة WebP خفيفة. شعارات العملاء في `assets/images/clients/`.
Photos are WebP. Client logos go in `assets/images/clients/`.
